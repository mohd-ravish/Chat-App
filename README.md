Built using:

- Front-End : React
- Back-End : NodeJs, ExpressJs, Socket
  
In the /Chat-Verse/client, run:
### `npm start`

In the /Chat-Verse/server, run:
### `nodemon server.js`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
